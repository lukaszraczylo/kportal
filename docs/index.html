Configuration

Simple YAML configuration

.kportal.yaml 
# Enable mDNS for .local hostnames
mdns:
  enabled: true

contexts:
  - name: production
    namespaces:
      - name: backend
        forwards:
          - resource: service/postgres
            protocol: tcp
            port: 5432
            localPort: 5432
            alias: prod-db  # Access via prod-db.local

      - name: frontend
        forwards:
          - resource: service/redis
            protocol: tcp
            port: 6379
            localPort: 6379
            # Auto-generates redis.local from resource name

Resource Types

  • pod/name - Direct pod
  • service/name - Service
  • deployment/name - Deployment
